# zhixuewang-python

## 安装:

### 使用 pip 安装(推荐)
```bash
pip install zhixuewang
```
### 下载 源码 安装
```bash
git clone https://github.com/anwenhu/zhixuewang
cd zhixuewang
python setup.py install
```

我们的文档在：[文档](https://zxdoc.risconn.com)，更多高级功能均将在那里介绍。
